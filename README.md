# SugarTrack

SugarTrack - это веб-приложение для управления здоровьем с поддержкой двух основных сценариев использования:

- Версия для диабета - дневник питания с подсчетом КБЖУ и хлебных единиц, трекинг измерений (вес, глюкоза, HbA1c, объемы тела), справочник гликемического индекса продуктов, учет лекарств
- Версия для РПП/эмоционального трекинга - эмоциональный дневник, дашборд настроения, срочная помощь в кризисных ситуациях

Проект включает полноценную систему аутентификации, персонализированные профили пользователей и функциональные модули для ежедневного контроля здоровья.

## Основные возможности

### Диабет-версия

#### Реализовано:
- **Дневник питания** - добавление приемов пищи с автоматическим подсчетом калорий, белков, жиров, углеводов
- **База продуктов** - 70+ продуктов с полной пищевой ценностью и гликемическим индексом
- **Профиль пользователя** - отображение личных данных, целей (вес, глюкоза, HbA1c) и фактических показателей с расчетом ИМТ
- **Страница измерений**:
    - История измерений (вес, глюкоза, HbA1c, объемы тела)
    - Визуализация данных с графиками (Chart.js)
    - Справочник гликемического индекса с поиском по продуктам
    - Калькулятор хлебных единиц (ХЕ)
- **Дашборд** - краткая статистика и быстрый доступ ко всем разделам

#### В разработке:
- Учет лекарств и инсулина
- Экспорт данных и отчеты
- Напоминания о приеме лекарств

### ED-версия

#### Реализовано:
- Базовая структура страниц
- Единый дизайн с диабет-версией

#### В разработке:
- Эмоциональный дневник с трекингом настроения
- Дашборд с аналитикой эмоционального состояния
- Модуль срочной помощи

## Технологический стек

### Backend:
- Node.js + Express - серверная часть
- MongoDB + Mongoose - база данных
- JWT (jsonwebtoken) - авторизация
- bcrypt - хеширование паролей
- express-rate-limit - защита от брутфорса

### Frontend:
- Bootstrap 5 - UI-фреймворк
- Chart.js - графики и визуализация данных
- Vanilla JavaScript - клиентская логика
- Кастомная темная дизайн-система (серо-фиолетовая палитра)

### Дополнительно:
- dotenv - управление переменными окружения
- nodemon - автоперезагрузка сервера при разработке

## Безопасность

- Хеширование паролей с bcrypt
- JWT токены для аутентификации
- Rate limiting на роуты авторизации
- Валидация всех входящих данных
- Защита от XSS через экранирование HTML
